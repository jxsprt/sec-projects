XSS Payload Playground

Create a safe environment to test and understand XSS payloads — reflected, stored, DOM-based, and filter evasion.

javascripthtmlxssweb-security

Learning Objectives

  • Understand the three XSS types: reflected, stored, and DOM-based
  • Build a Flask web app that safely demonstrates each XSS variant
  • Implement a payload library with categorized attack vectors
  • Learn filter evasion techniques — encoding bypasses, tag mutation, polyglots
  • Parse server logs to confirm which payloads executed successfully

Prerequisites

  • Python 3.8+ with flask installed
  • Basic HTML and JavaScript knowledge
  • Understanding of browser same-origin policy
  • A modern browser for testing (Chrome/Firefox DevTools recommended)
⚠ Ethics & Legal

This is a local-only playground — never deploy this to the public internet. All XSS testing should happen in an isolated local environment. XSS attacks against production websites are illegal without authorization.

Step 1: Flask App with Vulnerable Endpoints

Create a Flask application with deliberately vulnerable endpoints that demonstrate each XSS type. The playground will be fully self-contained.

from flask import Flask, request, render_template_string, make_response
import json, os

app = Flask(__name__)
app.config['SECRET_KEY'] = os.urandom(24)

# In-memory "database" for stored XSS demo
comments = []

# ---- REFLECTED XSS: search query echoed directly ----
@app.route('/')
def index():
    return render_template_string('''
<!DOCTYPE html>
<html>
<head><title>XSS Playground</title>
<style>
  body { font-family: monospace; max-width: 800px; margin: 2rem auto; padding: 0 1rem;
         background: #111; color: #eee; }
  .endpoint { border: 1px solid #333; border-radius: 8px; padding: 1rem; margin: 1rem 0; }
  .endpoint h3 { margin-top: 0; color: #58a6ff; }
  input, textarea { width: 100%; padding: 0.5rem; margin: 0.5rem 0;
                    background: #222; color: #eee; border: 1px solid #444; border-radius: 4px; }
  button { background: #238636; color: white; border: none; padding: 0.5rem 1rem;
           border-radius: 4px; cursor: pointer; }
  .payload { background: #161b22; padding: 0.5rem; border-radius: 4px; margin: 0.25rem 0; }
  code { color: #f0883e; }
</style>
</head>
<body>
  <h1>💣 XSS Payload Playground</h1>
  <p>Safe local environment — test payloads against vulnerable endpoints.</p>

  <div class="endpoint">
    <h3>1. Reflected XSS (Search)</h3>
    <form action="/search" method="GET">
      <input name="q" placeholder="Enter search term or payload...">
      <button type="submit">Search</button>
    </form>
  </div>

  <div class="endpoint">

    </form>
    <div id="comments">

        <div class="payload">{{ c | safe }}</div>
      {% endfor %}
    </div>
  </div>

  <div class="endpoint">
    <h3>3. DOM-Based XSS</h3>
    <p>Fragment-based — payload goes after the <code>#</code>:</p>
    <a href="/dom#&lt;img src=x onerror=alert(1)&gt;">Test DOM XSS →</a>
  </div>
</body>
</html>
    ''', comments=comments)

# ---- Reflected XSS endpoint ----
@app.route('/search')
def search():
    query = request.args.get('q', '')
    # VULNERABLE: directly embedding user input without escaping
    return f'<h2>Search results for: {query}</h2><p>No results found.</p><a href="/">← Back</a>'

# ---- Stored XSS endpoint ----
@app.route('/comment', methods=['POST'])
def add_comment():
    comment = request.form.get('comment', '')
    comments.append(comment)
    return '<script>window.location="/"</script>'

# ---- DOM-Based XSS endpoint ----
@app.route('/dom')
def dom_xss():
    return '''
<h2>DOM-Based XSS</h2>
<script>
  var hash = location.hash.slice(1);
  if (hash) {
    document.write("<p>Fragment content: " + hash + "</p>");
  }
</script>
<p>Add a payload after the # in the URL.</p>
<a href="/">← Back</a>
'''

if __name__ == '__main__':
    app.run(debug=True, host='127.0.0.1', port=5000)

Important: The | safe Jinja filter and raw string interpolation (f"...{query}") are what make these endpoints deliberately vulnerable. In production, you'd use auto-escaping and html.escape().

Step 3: Automated Payload Tester

Build a script that fires all payloads at each endpoint and records which ones execute — this is how you learn which vectors work against which contexts.

import requests
import urllib.parse

class XSSPayloadTester:
    def __init__(self, base_url="http://127.0.0.1:5000"):
        self.base_url = base_url
        self.results = []

    def test_reflected(self, payload):
        """Test against reflected XSS endpoint."""
        encoded = urllib.parse.quote(payload)
        url = f"{self.base_url}/search?q={encoded}"
        try:
            r = requests.get(url, timeout=5)
            return {
                'endpoint': 'reflected',
                'payload': payload[:60],
                'status': r.status_code,
                'response_length': len(r.text),
                'executed': payload in r.text,
            }
        except Exception as e:
            return {'endpoint': 'reflected', 'payload': payload[:60], 'error': str(e)}

    def test_stored(self, payload):
        """Test against stored XSS endpoint."""
        try:
            # Post the comment
            r = requests.post(f"{self.base_url}/comment",
                             data={'comment': payload}, timeout=5)
            # Fetch the homepage to see if it's reflected
            r2 = requests.get(self.base_url, timeout=5)
            return {
                'endpoint': 'stored',
                'payload': payload[:60],
                'status': r2.status_code,
                'executed': payload in r2.text,
            }
        except Exception as e:
            return {'endpoint': 'stored', 'payload': payload[:60], 'error': str(e)}

    def test_dom(self, payload):
        """DOM-based can't be detected server-side — note for manual testing."""
        return {
            'endpoint': 'dom',
            'payload': payload[:60],
            'note': 'DOM-based requires browser — test manually at /dom#',
        }

    def run_all(self, library):
        for category, payloads in library.items():
            if isinstance(payloads, dict):
                payloads = payloads.values()
            for payload in payloads:
                self.results.append(self.test_reflected(payload))
                self.results.append(self.test_stored(payload))
                self.results.append(self.test_dom(payload))

        # Print summary
        executed = [r for r in self.results if r.get('executed')]
        print(f"\nTested {len(self.results)} payloads — {len(executed)} executed successfully:")
        for r in executed:
            print(f"  [{r['endpoint']}] {r['payload']}")

tester = XSSPayloadTester()
tester.run_all(PAYLOAD_LIBRARY)

Step 4: Log Capture & Analysis

Extend the Flask app to log all requests, then analyze the logs to identify which payloads reached the server and triggered successful XSS.

# Add to Flask app (before routes)
import logging

logging.basicConfig(
    filename='xss_playground.log',
    level=logging.INFO,
    format='%(asctime)s [%(levelname)s] %(message)s'
)

@app.before_request
def log_request():
    """Log every incoming request for analysis."""
    logging.info(
        f"{request.method} {request.path} | "
        f"IP: {request.remote_addr} | "
        f"UA: {request.user_agent.string[:80]} | "
        f"Params: {dict(request.args)} {dict(request.form)}"
    )

# Log analyzer script (separate file: log_analyzer.py)
import re
from collections import Counter

def analyze_xss_logs(logfile='xss_playground.log'):
    with open(logfile) as f:
        lines = f.readlines()

    payloads_found = Counter()
    for line in lines:
        if '/search' in line:
            # Extract payload from query string
            match = re.search(r'q=([^&|\]]+)', line)
            if match:
                payloads_found[match.group(1)] += 1
        elif '/comment' in line:
            match = re.search(r"'comment': '(.+?)'", line)
            if match:
                payloads_found[match.group(1)[:80]] += 1

    print(f"Total XSS attempts: {sum(payloads_found.values())}")
    print(f"Unique payloads: {len(payloads_found)}")
    print("\nTop payloads:")
    for payload, count in payloads_found.most_common(10):
        print(f"  [{count}x] {payload[:70]}")

if __name__ == '__main__':
    analyze_xss_logs()
