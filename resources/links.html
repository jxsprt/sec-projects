Curated Resources

The best free and low-cost resources for learning cybersecurity — organized so you spend less time searching and more time learning.

🎯 Free Training & Labs

🎮 TryHackMe — Guided rooms from complete beginner to advanced. Start with the free tier. 🧠 Hack The Box Starting Point — Free beginner-friendly machines with walkthroughs. 🏆 PicoCTF — Carnegie Mellon's CTF designed for high school/college students. Great archives. 🗳 OverTheWire — Classic wargames. Start with Bandit (Linux basics) then Leviathan, Natas (web). 🎓 PortSwigger Web Security Academy — Free, comprehensive web security training by Burp Suite creators. 📦 VulnHub — Downloadable vulnerable VMs to practice on locally. 🛡 CyberDefenders — Blue team / DFIR challenges with PCAP and memory analysis.

📜 Certification Prep

📺 Professor Messer — Free Security+, Network+, A+ video courses. The gold standard. 📚 Jason Dion (Udemy) — Practice exams for Security+, CySA+, CASP+. Wait for Udemy sales ($15-20). 💪 TCM Security — Practical Ethical Hacking, PNPT prep. Affordable and hands-on. 🌐 INE / eLearnSecurity — eJPT, eCPPT training. Free tier has good basics.

🎬 YouTube Channels

☁ NetworkChuck — High-energy networking and cybersecurity content. Great for beginners. 🔍 John Hammond — CTF walkthroughs, malware analysis, practical tutorials. 🧠 IppSec — In-depth Hack The Box walkthroughs. Learn methodology from a master. 📡 David Bombal — Networking, GNS3 labs, ethical hacking interviews. 💻 LiveOverflow — Deep technical dives into binary exploitation, reverse engineering. 🎓 The Cyber Mentor — Full ethical hacking courses, PNPT-related content.

🏆 CTF Platforms

📅 CTFtime — Calendar of upcoming CTFs + team rankings and writeups. 🎮 PicoCTF Practice — Archived challenges from past competitions. 🧠 Hack The Box — Active machines, challenges, and competitive Pro Labs. 🎯 TryHackMe — Guided learning paths with built-in attack boxes.

📚 Books & Reading

📕 No Starch Press Security Catalog — Best technical publisher for security books. 📘 The Web Application Hacker's Handbook — Still relevant for web security fundamentals. 📗 Penetration Testing (Georgia Weidman) — Practical intro to pentesting methodology. 📙 Hacking: The Art of Exploitation — Deep C, assembly, and exploitation fundamentals.

🤝 Communities & Discord

💬 TryHackMe Discord — Active community, room discussions, and mentorship. 💬 Hack The Box Discord — Machine discussions (no spoilers), labs, and support. 🌐 r/cybersecurity — News, career advice, discussions. 🌐 r/netsec — Technical security research and papers. 🎓 r/netsecstudents — Student-focused security discussions and study groups.

📰 News & Threat Intel

📰 The Hacker News — Daily cybersecurity news. 💻 BleepingComputer — Breaking security news, ransomware coverage, tutorials. 🕵 Krebs on Security — Investigative security journalism by Brian Krebs. 📡 Feedly Cybersecurity — Aggregate RSS feeds from top security blogs.
